# Graph-Theoretic Analysis of the Human Vascular System

##  Overview
This project models the human vascular system as a directed graph to analyze blood flow and detect potential bottlenecks.

##  Key Features
- Applied Edmonds–Karp maximum flow algorithm
- Identified bottleneck vessels in the network
- Used centrality measures (betweenness, degree) to detect critical vessels
- Developed a risk scoring system for blockage prediction
- Simulated future blockage scenarios using capacity reduction

##  Concepts Used
- Graph Theory
- Network Flow Algorithms
- Centrality Analysis
- Simulation & Prediction

##  Visualization
- Graph-based network representation
- Color-coded risk levels (high, medium, low)
- Flow thickness to represent blood circulation
- Time-based simulation of blockage progression
